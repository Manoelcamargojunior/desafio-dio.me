Conseitos basicos de HTML

Voltar

HTML (Linguagem de Marcação de Hipertexto) é o código que você usa para estruturar uma página web e seu conteúdo. Por exemplo, o conteúdo pode ser estruturado em parágrafos, em uma lista com marcadores ou usando imagens e tabelas. Como o título sugere, este artigo fornecerá uma compreensão básica do HTML e suas funções, usada para definir a estrutura do seu conteúdo.


O que são tags e como funciona

Voltar

As tags HTML são elementos usados para marcar e formatar os conteúdos: textual ou visual em documentos HTML. Cada tag HTML tem um nome e uma sintaxe que determina como ela deve ser aberta e fechada.

Exemplos de tags:

  • < p> Tag de parágrafo < /p>
  • < h2>Tag de titulo < /h2>
  • < b>Tag de negrito< /b>
  • < i> Tag de itálico< /i>
  • < u> Tag de sublinhado< /u>

Resumo do projeto na plática

Voltar

HTML nao é uma linguagem de programação e sim um arquivo de marcação. Textos delimitados por nomes que o navegador consegue interpretar, cada nome destes delimitadores, tem uma função e comportamento específico. Os arquivos html podem ser visualizados em qualquer navegador (como Google Chrome, Safari, ou Mozilla Firefox). Geralmente um site é composto por diversas páginas HTML.
